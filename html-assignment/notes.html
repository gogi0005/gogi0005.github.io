These are some of the Important html tag for building a Website

"!doctype HTML" tag
is the first thing we add to our HTML document which defines what is the type of this document.
Heading "H1,H2,H3" tag
We can use the h1 tag for the most important heading of the page. For the next level of heading we have h2,h3,h4,h5,h6.
The browser will style each heading differently, stating with the biggest size for h1 then h2 and so on.
Pagaraphs "p" tag
For adding a paragraph of text we can use the tag. The browser will add a space before and after the content of the paragraph
Quote "q" tag
"q" a small quotation embedded within other content. "You become what you think"
Horizontal Line "hr" tag
"hr" represents a thematic break in the content. For example, a scene change or topic change.
